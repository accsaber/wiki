# AccSaber Ranking Criteria

## General
- 6 minute maximum time between first and last note
- No bad cut patterns
- Each hand must start on a downswing (down or down-diagonal)
- No curved sliders
- Note count minimum: 115
- No 3-wide dodge walls
- Min space before notes after bombs: 150ms
- Mods must not be required to play the map
- No audible off-time notes
- No hot starts

## True
- 2 minute minimum time between first and last note
- NJS limit: 12 NJS
- Max peak SPS counting doubles as one swing: 1.75
- Max average SPS counting doubles as one swing: 1.5
- No stacks
- No sliders
- No windows
- Resets must be consistent
- No wristrolls
- Min time after notes before bombs: 500ms

## Standard
- 1:45 minute minimum time between first and last note
- NJS limit: 16 NJS
- Max peak SPS: 6.25
- Max average SPS: 4
- No DDs
- No triangles
- No windows
- No sliders
- No towers
- No wristrolls
- Min time after notes before bombs: 250ms

## Tech
- 1:45 minute minimum time between first and last note
- NJS limit: 16 NJS
- Max peak SPS: 6.25
- Max average SPS: 4
- No DDs
- Min time after notes before bombs: 250ms
